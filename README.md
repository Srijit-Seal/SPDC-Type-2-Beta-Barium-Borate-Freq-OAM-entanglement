# SPDC-Type-2-Beta-Barium-Borate-Freq-OAM-entanglement
Abstract
Spontaneous Parametric Down Conversion (SPDC) is a non-linear χ(2) process where
one (pump) photon spontaneously down-converts(energy) into 2 correlated photons
(signal and idler). This process is key to the generation of entangled photonic qubits
and qudits. It has possible applications in quantum key distribution and quantum
information processing.
The key challenge that is being tackled here is that this process is stochastic even more
so for entanglement of 2 degrees of freedom such as Orbital Angular Momentum and
Frequency Hybrid Entanglement. The downconverted photon pair generation depends
not only on the output field amplitudes but also on the vacuum fluctuations of the
signal and idler beam. These vacuum fluctuations are fundamentally random. Very
few photons that go into the nonlinear crystal get down converted into signal and idler
photons. To increase the efficiency of the experiment the pump, signal and idler beams
must be precisely phase-matched. To obtain a desirable maximally entangled state, one
can very experiment parameters such as pump waist or crystal structure. Although to
vary these parameters in a way as to obtain the desirable state is a challenge. So,
this project aims at designing the experiment simulation for being used in a machine
learning model to learn this experiment and obtain the optimum parameters such as
pump waist, structure of the crystal, design of post processing optics such as factors of
rotating q plates, that will give us the desired maximally entangled state.
